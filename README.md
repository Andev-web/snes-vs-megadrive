# 🎮 SNES vs Mega Drive

Pequeño proyecto web estático creado con **HTML y CSS** inspirado en la clásica guerra de consolas de los **16 bits**.

La página compara **Super Nintendo** y **Sega Mega Drive**, mostrando sus características, estilo y algunos de sus juegos más representativos.

## 🕹️ Características

* Diseño retro inspirado en los **videojuegos de los 90**
* Tipografías pixel art
* Comparativa visual entre las consolas
* Tarjetas con información de cada sistema
* Estética **16-bit / arcade**

## 🧰 Tecnologías utilizadas

* HTML5
* CSS3
* Google Fonts (Press Start 2P, VT323)
